BigQuery is a cloud data warehouse used for storing and analysing large datasets. It allows users to run SQL queries on structured data without managing database servers manually. This makes it suitable for business intelligence tasks where organisations need to explore customer, product, order, and sales information.
In an eCommerce environment, a data warehouse can combine data from different parts of the business. Customer records, product catalogues, order transactions, web events, and marketing data can be analysed together. This helps decision makers identify sales trends, high-performing products, customer behaviour patterns, and operational issues.
Business intelligence dashboards are useful because they turn query results into visual reports. Instead of reading raw tables, users can view charts, scorecards, and summaries. In this coursework, BigQuery is used to query the data warehouse, while Looker Studio is used to visualise business insights. This connection between data storage, SQL analysis, and dashboard reporting shows how cloud services support real business decision making.
